### Installation

Using [vim-pathogen](https://github.com/tpope/vim-pathogen)
```bash
cd ~/.vim/bundle
git clone https://github.com/rudes/vim-java.git
```

Using [Vundle](https://github.com/VundleVim/Vundle.vim)
```bash
Plugin 'rudes/vim-java'
```
